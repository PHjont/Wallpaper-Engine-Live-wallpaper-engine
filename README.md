# Wallpaper Engine

> **Wallpaper Engine** — The most popular tool to bring animated and interactive wallpapers to your Windows desktop.

[![Stars](https://img.shields.io/github/stars/PHjont/Wallpaper-Engine-Live-wallpaper-engine)](https://github.com/PHjont/Wallpaper-Engine-Live-wallpaper-engine) [![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](https://opensource.org/licenses/MIT)

**Wallpaper Engine** is a powerful and easy-to-use application that lets you set beautiful animated, live, and interactive wallpapers on your desktop. Browse thousands of community creations, customize them, and make your screen truly unique.

## ❗️ Troubleshooting
| Problem                    | Solution                                      |
|----------------------------|-----------------------------------------------|
| App doesn't start          | Run as Administrator                          |
| Low FPS on desktop         | Lower wallpaper quality in settings           |
| Black screen               | Restart Windows Explorer                      |
| Antivirus warning          | Add the folder to exclusions                  |
| Wallpapers not playing     | Check GPU drivers and hardware acceleration   |
